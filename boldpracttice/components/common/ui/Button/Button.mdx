# Button

汎用的なボタンコンポーネントです。buttonタグとaタグの両方に対応しています。

## 機能

- **variant**: primary, secondary, outline の3種類
- **size**: small, medium, large の3サイズ
- **as**: 'button' または 'a' でタグを切り替え
- **color**: カスタム背景色を指定可能
- **8pxグリッド準拠**: すべてのスペーシングは8pxの倍数

## 使用例

### ボタンとして使用

```tsx
import { Button, BUTTON_VARIANTS, BUTTON_SIZES } from '@/components/common/ui/Button';

<Button 
  variant={BUTTON_VARIANTS.PRIMARY} 
  size={BUTTON_SIZES.MEDIUM}
  onClick={() => console.log('clicked')}
>
  Click me
</Button>
```

### リンクとして使用

```tsx
<Button 
  as="a" 
  href="/register"
  variant={BUTTON_VARIANTS.PRIMARY}
  size={BUTTON_SIZES.SMALL}
>
  Register
</Button>
```
