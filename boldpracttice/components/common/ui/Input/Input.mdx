# Input

共通のInput要素コンポーネントです。

## 使い方

```tsx
import { Input, INPUT_TYPES } from '@/components/common/ui/Input';

<Input
  label="メールアドレス"
  inputType={INPUT_TYPES.EMAIL}
  placeholder="you@example.com"
  required
/>
```

## Features

- サイズバリエーション (small, medium, large)
- 複数のinputタイプ対応 (text, email, password, number, tel, url, date)
- エラー表示機能
- ラベル表示
- 必須マーク自動表示
- 無効化状態のサポート
